Java Thread lifecycle, including the states in the Java Thread state machine, as well as details on how Java Threads are started and stopped.

A Java thread can exist in one of six states:

NEW - Created but not yet started

RUNNABLE - Executing in the JVM

BLOCKED - Waiting for a monitor lock

WAITING - Waiting indefinitely for another thread

TIMED_WAITING - Waiting for a specified time

TERMINATED - Completed execution
